Experiences

Deepfake Video Detection System

I built a DeepFake Video Detection web app that identifies whether a video is real or fake. The system extracts video frames, detects faces, and uses a deep learning model built with PyTorch (ResNeXt + LSTM) for classification. Flask handles the backend and connects with a simple HTML/CSS/JS frontend for user interaction. The app returns the result with a confidence score, making DeepFake detection quick and accessible.

Personalized Travel Planner

Personalized Travel Planner is a web-based application that helps users create customized travel itineraries based on their preferences. Users can register, fill out a travel questionnaire, and get suggestions for hotels, homestays, and tourist spots. The project uses HTML, CSS, JavaScript for the frontend, PHP for backend logic, and MySQL for database management. It simplifies the trip planning process by offering budget-friendly and destination-specific travel options..

E-Hospitality

This project is a Django-based E-Hospitality web application designed to streamline hospital management processes. It features user authentication with role-based access for Admin, Doctor, and Patient, ensuring secure and personalized interactions. The system allows patients to book appointments, while doctors can manage schedules and view patient records. It emphasizes a user-friendly interface, robust backend logic, and seamless database integration. The goal is to simplify healthcare workflows through an efficient and responsive web platform.
